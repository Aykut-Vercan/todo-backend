# Todo App Backend

This is the backend of the **Todo Application**, built with **Spring Boot**, **MySQL**, and **JWT authentication**.  
It provides secure RESTful APIs for user registration, login, and todo management.  
The backend is deployed on **Railway** and designed to work seamlessly with the React frontend hosted on **Vercel**.
